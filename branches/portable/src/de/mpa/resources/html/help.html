Introduction

The MetaProteomeAnalyzer (MPA) Portable constitutes a stand-alone software pipeline to analyze metaproteomics (and also proteomics) data sets. The software is being developed in the Java programming language and uses a neo4j graph database backend for flexible and user-defined data querying of proteomics results.

For more information, please visit the MPA web pages at the MPA Homepage

Current release version: 1.0.1

MPA Portable Application

The MPA Portable comes with the following key features:

  • Intuitive graphical user interface
  • Project and experiment management
  • X!Tandem, OMSSA and MASCOT support
  • Stand-alone processing and protein identification pipeline
  • Top-down analysis
  • Label-free quantification methods
  • Graph database-driven
  • Interactive Overview (Bar and Pie Charts)
