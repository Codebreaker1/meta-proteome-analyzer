MetaProteomeAnalyzer

The Meta-Proteome-Analyzer constitutes a distributed computing software pipeline to analyze (meta-) proteomics datasets.
Current developer version: 0.5

The client comes with the following key features:

  • SQL database back-end
  • Algorithms workload server-side
  • Intuitive graphical user interface
  • Project and experiment management
  • Top-down analysis
  • Label-free quantification methods
